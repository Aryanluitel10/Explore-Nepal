# 🇳🇵 Explore Nepal

A full-featured Nepal travel guide web app built with **React**, **Vite**, **Tailwind CSS**, and **Framer Motion**.

Discover Nepal's top destinations, traditional food, rich culture, ancient history, travel tips, photo gallery, and latest news — all in one beautiful web app.

---

## Features

- **Destinations** — Explore Kathmandu, Pokhara, Bhaktapur, Lumbini, Chitwan, and more
- **Food** — Discover traditional Nepali cuisine like momos, dal bhat, sel roti, and more
- **Culture** — Learn about festivals, traditions, and Nepali way of life
- **History** — Journey through Nepal's ancient, medieval, and modern history
- **Travel Guide** — Practical tips for visiting Nepal
- **Gallery** — Beautiful photo collection of Nepal
- **News** — Latest updates from Nepal
- **Multi-language support** — Built-in i18n support

---

## Tech Stack

| Technology | Purpose |
|---|---|
| React + Vite | Frontend framework & build tool |
| Tailwind CSS | Styling |
| Framer Motion | Animations |
| React Router | Page navigation |

---

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```
